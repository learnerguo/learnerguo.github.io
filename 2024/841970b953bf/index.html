tomcat版本：9.0.84
jdk版本：17

下载 tomcat9 源码

下载tomcat9
选择下载页面最下方的 Source Code Distributions 下的 zip 包

源码导入idea之前的准备

  1. 解压下载的源码，得到目录 apache-tomcat-9.0.84-src
  2. 进入 apache-tomcat-9.0.84-src 目录，创建一个 pom.xml 文件，pom 文件内容会在下方给出
  3. 在 apache-tomcat-9.0.84-src 目录中创建 source 文件夹
  4. 将 conf、webapps 目录移动到刚刚创建的 source 文件夹中

导入idea

idea 导入 tomcat9 源码时只需要选择 apache-tomcat-9.0.84-src 目录或者选择该目录的的 pom.xml 文件即可
