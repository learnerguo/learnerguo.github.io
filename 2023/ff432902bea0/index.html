Window下安装mysql8解压版

发表于2023-12-13|更新于2023-12-13|存储Mysql
|字数总计:597|阅读时长:2分钟|阅读量:
  • 下载Mysql8， 选择 Windows (x86, 64-bit), ZIP Archive 这个
  • 解压到某个目录，比如 E:\config\mysql-8.2.0-winx64， 这就是 Mysql 安装目录
  • 在 Mysql 安装目录下新建 data 目录用来存放数据，即 data 目录全路径为 E:\config\mysql-8.2.0-winx64\data
  • 在 Mysql 安装目录下新建配置文件 my.cnf, 内容如下
  • 使用管理员身份打开 cmd, 进入 Mysql 安装目录下的 bin 目录

初始化完成后在 cmd 窗口会出现 A temporary password is generated for root@localhost: ,mDQ>kFHI65w 字样(密码生成的不同)， 复制保存这个密码

  • 继续在当前 cmd 窗口执行 mysqld --install 命令，这个命令是将 Mysqld 程序作为安装为 Windows 服务， 这样就可以在系统服务中看到 Mysqld 了
  • 执行 net start MySQL 就可以启动 Mysql 服务了
  • 执行 mysql -u root -p 命令，然后复制上面拷贝出来的 ,mDQ>kFHI65w 密码，直接用这个密码去登录就行了
  • 修改密码为 admin, 后续就可以使用 root 账号和 admin 密码登录了

问题
执行 net start mysql 时出现 无法启动服务，原因可能是已被禁用或与其相关联的设备没有启动。
此时可以先卸载 Mysql 服务再重试安装
