дҪҝз”Ё nacos2.2.3 зүҲжң¬

  1. д»Һ githubдёӢиҪҪnacosжәҗз Ғ
  2. еҲҮжҚўеҲ° 2.2.3 зүҲжң¬
    1. git checkout -b 2.2.3 2.2.3пјҢ nacos жәҗз Ғдёӯ 2.2.3 жҳҜ tag, иҜҘе‘Ҫд»ӨжҳҜд»ҺеҜ№еә” tag еҲӣе»әеҲҶж”Ҝ

ж·»еҠ  postgres дҫқиө–

еңЁ nacos зҡ„ж №зӣ®еҪ•дёӢзҡ„ pom.xml дёӯж·»еҠ еҰӮдёӢеҶ…е®№

еңЁ config жЁЎеқ—дёӢзҡ„ pom.xml дёӯж·»еҠ еҰӮдёӢеҶ…е®№

йҖҡиҝҮ spi жңәеҲ¶ж·»еҠ  postgres зӣёе…іж”ҜжҢҒ

жүҫеҲ° plugin жЁЎеқ—пјҢйҮҢйқўиҝҳеҢ…еҗ«дёҖдёӘ datasource жЁЎеқ—пјҢзӣ®еҪ•з»“жһ„еҰӮдёӢ
гҖҗdatasourceгҖ‘жЁЎеқ—зҡ„зӣ®еҪ•и·Ҝеҫ„дёәпјҡ nacosжәҗз ҒжүҖеңЁзҡ„зӣ®еҪ•дҪҚзҪ®\nacos\plugin\datasource

еңЁ com.alibaba.nacos.plugin.datasource.impl еҢ…дёӢе·Із»Ҹжңүзҡ„дёӨдёӘе®һзҺ°еҢ…жҳҜ mysql е’Ң derby, еҸҜд»ҘжӢ·иҙқдёҖдёӢ mysql ж•ҙдёӘеҢ…еӨҚеҲ¶еҲ° mysql еҗҢзә§зӣ®еҪ•дёӢе№¶дё”ж”№еҗҚдёә postgres, еҢ…йҮҢйқўзҡ„жҜҸдёӘзұ»зҡ„еҗҺзјҖж”№дёә Postgres, ж”№е®ҢеҗҺ postgres зӣ®еҪ•дёӢзҡ„зұ»жңүеҰӮдёӢ 9 дёӘ

иҝӣиЎҢдёҠиҝ°ж“ҚдҪңеҗҺеҸӘжҳҜе°Ҷ mysql зӣ®еҪ•ж•ҙдёӘжӢ·иҙқдәҶдёҖд»Ҫдҝ®ж”№зӣ®еҪ•еҗҚе’Ңзұ»еҗҚпјҢдҪҶжҳҜйҮҢйқўзҡ„еҶ…е®№е…¶е®һжІЎжңүеҸҳеҢ–пјҢжҲ‘д»¬д»Ҙ ConfigInfoMapperByPostgreSqlпјҲmysql зӣ®еҪ•дёӢпјү дёәдҫӢпјҢзңӢзңӢйңҖиҰҒдҝ®ж”№д»Җд№ҲеҶ…е®№

дҝ®ж”№зӮ№1
иҝҷйҮҢзҡ„дҝ®ж”№жҳҜеӣ дёә mysql е’Ң postgres зҡ„еҲҶйЎөиҜӯжі•дёҚеҗҢ
mysql еҲҶйЎөпјҡLIMIT startRow,pageSize
postgres еҲҶйЎөпјҡLIMIT pageSize OFFSET startRow

дҝ®ж”№зӮ№2
getDataSource() ж–№жі•иҝ”еӣһзҡ„жҳҜж•°жҚ®жәҗзұ»еһӢпјҢmysql дёӯиҝ”еӣһзҡ„жҳҜ DataSourceConstant.MYSQLпјҢзңӢдёҖдёӢ DataSourceConstantзұ»д»Јз Ғ

й»ҳи®ӨеҸӘжңүдёӨдёӘпјҢзҺ°еңЁж–°еўһдёҖдёӘ postgres зұ»еһӢпјҢеҸҳжҲҗ

дёӨеӨ„йғҪдҝ®ж”№е®ҢжҲҗд№ӢеҗҺпјҢpostgres зӣ®еҪ•дёӢеҜ№еә”зҡ„ ConfigInfoAggrMapperByPostgreSql зұ»з»ҸиҝҮдҝ®ж”№еҗҺеҸҳжҲҗеҰӮдёӢеҪўејҸ

ж·»еҠ spiзұ»й…ҚзҪ®

дёӢйқўдјҡеҲ—еҮәдҝ®ж”№еҗҺжүҖжңү postgres зӣ®еҪ•дёӢжүҖжңүзұ»д»Јз ҒпјҢдёҚиҝҮиҝҷйҮҢе…Ҳе°Ҷе…¶д»–йңҖиҰҒдҝ®ж”№зҡ„й…ҚзҪ®ж”№е®Ң

жүҫеҲ° plugin/datasource жЁЎеқ—дёӢ resources/META-INF/services зӣ®еҪ•пјҢиҜҘзӣ®еҪ•дёӢжңүдёҖдёӘй…ҚзҪ®ж–Үд»¶еҗҚдёә com.alibaba.nacos.plugin.datasource.mapper.Mapper
е°ҶеҰӮдёӢеҶ…е®№ж·»еҠ еҲ°иҝҷдёӘж–Үд»¶дёӯ

дҝ®ж”№ db й…ҚзҪ®

еңЁ console жЁЎеқ—дёӢзҡ„ application.properties ж–Үд»¶дёӯж·»еҠ еҰӮдёӢй…ҚзҪ®

postgres зӣ®еҪ•дёӢжүҖжңүзұ»
