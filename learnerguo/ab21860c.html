Hexo Next主题支持 mermaid

  1. 进入博客根目录，执行安装插件命令 npm install hexo-filter-mermaid-diagrams --save
  2. 打开 Next 主题下的 _config.yml 文件，将 mermaid 的配置改成 true
  1. 重新生成部署即可
